# Sample Documents

This folder contains sample markdown files for testing the document upload functionality.

## Using Sample Documents

1. Go to `/upload` page
2. Click "Upload Document" button
3. Drag and drop or select `boulder-redemption-2025.md`
4. Fill in the metadata:
   - County: Boulder
   - Title: Redemption Guidelines 2025
   - Year: 2025
5. Review the preview showing detected headers
6. Click "Upload Document"

## File Structure

Sample documents should follow this structure:

```markdown
# Main Title

## Section 1
Content...

## Section 2
Content...

### Subsection 2.1
Details...
```

## Tips

- Use clear, descriptive headers (H1, H2)
- Break content into logical sections
- Include structured information that's easy to chunk
- Keep file size under 5MB
